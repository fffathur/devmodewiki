# PPSSPP Setup Guide

This guide shows you how to set up PPSSPP on Xbox using either an **external USB drive (recommended)** or **internal storage**.

If you have not prepared your USB for emulators yet (NTFS format and correct permissions), follow the **USB section** in the [Xbox Developer Mode Setup Guide](/docs/xbox-setup/xbox-developer-mode-setup) first, then come back here.

## PPSSPP External USB Users

:::caution
Make sure your USB is NTFS and has **ALL APPLICATION PACKAGES -> Full Control**, otherwise games will not load. Use the USB section in the [Xbox Developer Mode Setup](/docs/xbox-setup/xbox-developer-mode-setup) guide.
:::

## Download PPSSPP

Get PPSSPP from [Emulators on Emulation Revival Store](https://emulationrevival.github.io/xbox-dev-mode/emulators.html#ppsspp) (or the [PPSSPP UWP release](https://www.ppsspp.org/download/) the store mirrors). The download is a **`ppsspp_uwp.zip`**: extract it on your PC, then use Device Portal **My games & apps - Add** to sideload the **`PPSSPP*.appx`** inside (add any dependency packages the zip includes if Device Portal asks).

Once your USB has been prepared using the Dev Mode USB steps (formatted to NTFS and with the correct permissions), plug it into your Xbox, open PPSSPP, and it will ask whether to set the PSP folder on **External (USB)** or **Internal (app storage)**.

**Create or Choose a PSP folder = External USB**

**Use App Private Data = Internal Storage**

![PSP Main Guide 2](/img/PPSSPP/pspmainguide2.png)

But If you accidentally choose Use App Private Data then No Problem! Just go to Settings > System > Click Memory Stick Folder then Choose Create or Choose a PSP folder

Once you choose "Create or Choose a PSP folder" then PPSSPP will create Folders for you automatically on USB then it should be like this in the Picture:

## Games

to upload your Games:

**Dev portal > File explorer > LocalAppData > PPSSPP > Localstate > Create a games folder here**

Your Games should be ISO or CHD files.

You can upload them as .zip files, and the dev portal will prompt you to unzip automatically.

## Textures, Cheats and Others

For Custom Textures, cheats etc. Upload them to their respective folders within:

**Dev portal > File explorer > LocalAppData > PPSSPP > Localstate > PSP**
